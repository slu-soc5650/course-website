Course Resources

This page is meant to augment the core documents by providing easy access to online elements of the course. The Syllabus contains a full list of resources, including those not available online.

Core Documents

There are two documents for this course that spell out what is required:

  • Syllabus
  • Reading List

Course Website

This course website will be updated regularly throughout the semester. I’ll add a page on the menu for each week, and include links to lecture slides, YouTube videos, and notes about specific topics. The course website also contains specific pages for the final project and two index pages cataloguing topics covered and R packages used. Enteries on both index pages include links to the relevant week or weeks where the item is covered.

Readings

Throughout the semester, there will be assigned readings from a number of different books. These include a number that you will have to buy (see the Syllabus and the Getting Started page) and some that you can access online. The books available online include:

  • R for Data Science
  • Data Visualization for Social Science
  • Sociospatial Data Science

Other articles and chapters not included in these books will be posted on our course’s electronic reserves site, and the password is posted in Slack on the #helpdesk-coursework channel.

GitHub

All course materials will be posted to repositories on our course GitHub Organization. Each student will have their own private repository in the same organization for submitting course work.

Slack

Slack is a communication platform for groups. We’ll use Slack this semester to stay in touch, provide updates on the course, and as a space for virtual office hours.
